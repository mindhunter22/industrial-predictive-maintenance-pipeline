## 💡 Why this project matters

Industrial systems generate large volumes of data, but failures are rare and costly.

This project focuses on:
- Detecting rare failure events (class imbalance problem)
- Optimizing recall to avoid missing critical failures
- Providing actionable insights via dashboards

The goal is to simulate how a real company would monitor machine health and prevent downtime.

## 🧠 Skills demonstrated

- Data Engineering: data cleaning, pipeline design, SQL integration
- Machine Learning: classification, imbalance handling, threshold tuning
- Data Analysis: exploratory analysis, anomaly detection
- Business Intelligence: Power BI dashboards and KPIs
- Problem Solving: real-world industrial use case
